## Featured projects

A browser-based accounting reconciliation tool for comparing movements from two different sources, proposing matches and reviewing unresolved records without sending financial data to a server.

**HTML · CSS · JavaScript · SheetJS · Web Workers**

### [MASA](https://andresgonzalez.netlify.app/masa/)

An engineering-inspired health analytics platform that treats body weight as a measurable system. It combines weight records, energy balance and observed trends to evaluate whether the mathematical model still represents reality.

**HTML · CSS · JavaScript · Data modeling**
